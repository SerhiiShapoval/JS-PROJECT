OrbitControls – це контролер камери в бібліотеці Three.js, який дозволяє користувачеві керувати положенням та орієнтацією камери у тривимірному просторі.

виражений у радіанах. Якщо значення цієї властивості встановлено, камера не може обертатися вище за заданий кут.

виражений у радіанах. Якщо значення цієї властивості встановлено, камера не може обертатися нижче за заданий кут.

SpotLight у Three.js – це джерело світла, яке випромінює світло у певному напрямку, утворюючи конус світла.
Він працює схожий на ліхтар, який можна направити на певний об'єкт у сцені.

useTexture – це функція або метод, який використовується у різних графічних API (наприклад, OpenGL, WebGL, DirectX) для завантаження та використання текстур візуалізації.
Вона використовується для прив'язування текстури до певного об'єкта або поверхні візуалізації.

Компонент Float використовується для створення анімацій у 3D-сценах React за допомогою бібліотеки Three.js.
Цей компонент дозволяє створювати плавні анімації, змінюючи числове значення від 0 до 1.

Тег ambientLight використовується в бібліотеці Three.js для створення джерела навколишнього світла у 3D-стіні.
Цей тип джерела світла не має напряму і не проектує тіні, а просто висвітлює всі об'єкти в сцені однаково, наче світло йде з усіх боків одночасно.

Тег directionalLight використовується у бібліотеці Three.js для створення спрямованого джерела світла у 3D-сцені.
Цей тип джерела світла має певний напрямок та проектує тіні, що дозволяє створювати більш реалістичне освітлення у 3D-сценах.

icosahedronGeometry – це геометричний тег у бібліотеці Three.js, який використовується для створення геометрії ікосаедра, тобто багатогранника з 20 гранями.

Додаткове пояснення по тегу mesh:
Mesh у бібліотеці Three.js використовується для створення об'єктів 3D-сцени. Цей тег є основним компонентом для відображення 3D-моделей і формує основу для налаштування їхнього зовнішнього вигляду та поведінки.
Щоб створити об'єкт із використанням тега mesh, потрібно вказати його геометрію та матеріал.
Геометрія визначає форму об'єкта, а матеріал визначає його зовнішній вигляд та властивості, такі як колір, текстури та блиск. У mesh можна також задати позицію, обертання та масштаб об'єкта у 3D-просторі.

Компонент Points з @react-three/drei є 3D-компонентом, який дозволяє відображати набір точок (часто званих "частинками") у сцені Three.js. Вона може бути використана для створення ефектів, таких як дощ, сніг, зоряне небо і т.д.

Основні пропси компонента Points включають:

    -points: масив точок, який потрібно відобразити.
    -positions: позиція точок у тривимірному просторі Це може бути тривимірний масив, одномірний масив або функція, яка приймає індекс точки як аргумент і повертає позицію.
    -color: колір точок. Це може бути задано як один колір для кожної точки або один колір для всіх точок.
    -size: розмір точок у пікселях
    -alphaMap: текстура, що визначає прозорість точок.
    -transparent: визначає, чи прозорі точки.
    -depthTest: визначає, чи мають точки брати участь у тесті глибини.
    -depthWrite: визначає, чи можуть точки записувати значення глибини у буфер глибини.
    -Blending: визначає, як точки змішуються з існуючими пікселями в буфері кадру.
    -rotation: Визначає обертання точок в тривимірному просторі.
    -material: матеріал, який використовується для відображення точок. За замовчуванням використовується PointsMaterial із Three.js.
